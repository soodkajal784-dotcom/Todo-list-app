# 📝 Todo App

A simple and clean Todo List web app built using HTML, CSS, and JavaScript.
It allows users to add, edit, delete, and mark tasks as completed.


🚀 Features

* ➕ Add new todos
* ✅ Mark todos as complete
* ✏️ Edit todos (double-click)
* 🗑️ Delete todos
* 💾 Saves data using Local Storage (data persists after refresh)


 🛠️ Tech Stack

* HTML
* CSS
* JavaScript (Vanilla JS)


 📂 Project Structure


todo-app/
 ├── index.html
 ├── style.css
 ├── script.js
 └── README.md



 ▶️ How to Run

1. Download or clone the repo:


git clone https://github.com/YOUR_USERNAME/todo-app.git


2. Open `index.html` in your browser
